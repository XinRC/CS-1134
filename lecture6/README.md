### ***Dynamic Arrays***
- Arrays are fixed-size data structures.
  - Stores a sequence of values
  - Stores contiguously (together in memory)
  - Requires all elements to be the same size

```python

print(len(lst)) # will output 5
print(lst[4]] # will output 50
```
